# hyunbenny's blog

[just-the-docs](https://github.com/just-the-docs/just-the-docs) 테마 기반 기술 블로그입니다.

- 컬러: 라이트/다크 커스텀 스킴 (`_sass/color_schemes/hyunbenny*.scss`)
- 글 작성: `docs/<카테고리>/` 아래에 md 파일 추가 (front matter는 기존 글 참고)
- 로컬 실행: `bundle install && bundle exec jekyll serve`
- 배포: master 푸시 시 GitHub Actions가 자동 빌드/배포
